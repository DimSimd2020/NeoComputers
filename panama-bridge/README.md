# panama-bridge

Shared contract zone between native C++ and Java.

## What belongs here

- stable headers for native/FFM interop
- Java-side interface definitions for bridge access
- memory layouts, handle types, enums, and ABI-safe structs
- versioned contract notes

## What does not belong here

- emulator implementation code
- NeoForge gameplay logic
- UI behavior
- experimental APIs that are not intended to be frozen
